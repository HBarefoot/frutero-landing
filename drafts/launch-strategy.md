# Frutero Launch Strategy — Phase 1

## Goal: 10 cloud subscribers

### Week 1: Launch
- [ ] Post to Hacker News (Show HN)
- [ ] Post to r/selfhosted, r/homelab, r/MushroomGrowers
- [ ] Tweet from personal account linking to GitHub
- [ ] Reply to relevant HN comments and Reddit threads

### Week 2: Direct Outreach
- [ ] Etsy mushroom grow kit sellers (target: 30)
- [ ] Instagram growers with "DM for bulk" or "grow kits" (target: 50)
- [ ] Local farmers market mushroom vendors (South Florida)
- [ ] Reddit cold messages to posters in growing communities

### Week 3: Content
- [ ] Medium post: "I Automated My Mushroom Farm With $40 of Raspberry Pi Hardware"
- [ ] YouTube demo: 3-min walkthrough of dashboard
- [ ] Cross-post to IndieHackers

### Email capture
- Use Formspree (free tier: 50 submissions/month)
- Capture before they leave the page (after pricing, before FAQ)
- Trigger: scroll depth + exit intent (optional, keep simple for now)

### Metrics
- Landing page views
- GitHub stars (conversion indicator)
- Cloud signups
- Email signups
- Source attribution (UTM params on posts)
